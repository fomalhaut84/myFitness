# 레이아웃 & 네비게이션 디자인 노트

## 미적 방향

**산업적 미니멀리즘 × 운동 정밀성**

사이드바는 고급 스포츠 워치 인터페이스 느낌: 어둡고, 정밀하고, 기능적.
모든 요소가 목적을 가지며, 장식적 요소 없이 깔끔한 기술적 실행.

## 주요 디자인 결정

### 활성 상태: 펄스 바
- 활성 네비게이션 항목 왼쪽에 2px 수직 액센트 바
- 그린(#22c55e) — 앱 전체에서 사용되는 동일 액센트
- 심박 모니터 라인을 연상시키는 피트니스 메타포

### 타이포그래피
- 브랜드 "myFitness": Geist Mono (모노스페이스, 기술적 느낌)
- 네비게이션 라벨: 시스템 산세리프, 13px, 넓은 자간
- 작고 정밀한 텍스트 — 계기판 수치 같은 느낌

### 컬러 팔레트 (globals.css 기준)
- 배경: #0a0a0a (거의 블랙)
- 사이드바 테두리: #1e1e1e (미세한 분리감)
- 활성 항목 배경: #161616 (미묘한 부각)
- 활성 아이콘: #22c55e (액센트 그린)
- 비활성 텍스트: #737373 → 호버 #a3a3a3
- 활성 텍스트: #ededed (밝음)

### 공간 구성
- 사이드바 너비: 240px (w-60) — 컴팩트하지만 읽기 쉬움
- 네비게이션 항목 패딩: 10px — 편안한 터치 영역
- 항목 간 0.5px 간격 — 촘촘한 계기판 느낌
- 브랜드 영역에 여유 공간 (pt-7 pb-8)

### 모바일 전략
- 상단 고정 헤더(h-12)에 햄버거 메뉴
- 사이드바가 왼쪽에서 슬라이드 + 배경 블러
- 헤더에 bg 투명도 + backdrop-blur로 깊이감

### 하단: 싱크 상태
- 작은 펄싱 그린 점 + "Garmin 연동됨"
- 항상 표시 — 데이터 파이프라인 정상 확인용
- 11px 텍스트, 절제된 색상(#525252) — 참고용, 시선 분산 최소화

## 반응형 브레이크포인트
- `md` (768px): 사이드바 고정 ↔ 사이드바 숨김 + 모바일 헤더
