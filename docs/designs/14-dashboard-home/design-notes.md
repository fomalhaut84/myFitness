# 대시보드 홈 디자인 노트

## 구성

### 오늘 요약 카드 (4개, 2×2 모바일 / 4×1 데스크톱)
- 걸음 수, 안정시 심박, 수면 점수, 바디배터리
- 각 카드: 아이콘 + 라벨(uppercase, 11px) → 큰 수치(24px, mono) → 전일 대비 변화
- 양수 변화: 초록(#22c55e) + 위 화살표
- 음수 변화: 빨강(#ef4444) + 아래 화살표 (심박은 내려가면 초록)
- 단위(bpm 등)는 작은 크기로 수치 옆에

### 주간 미니차트
- 7일 바 차트, 높이 80px
- 오늘 바: 진한 그린, 나머지: 그린 투명도 단계
- 요일 라벨 하단

### 최근 활동 목록
- 최근 3~5건
- 아이콘(운동 타입별 색상) + 이름/시간 + 거리/페이스 또는 칼로리
- 러닝: 그린 (#22c55e), 근력: 블루 (#60a5fa)
- 페이스: min'sec" 형식 (모노스페이스)

## 컬러 규칙
- 심박 변화: 내려가면 좋음 → 초록, 올라가면 → 빨강
- 바디배터리 변화: 올라가면 좋음 → 초록, 내려가면 → 빨강
- 걸음 수/수면: 올라가면 → 초록, 내려가면 → 빨강
