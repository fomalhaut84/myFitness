# 혈압 트래킹 UI 디자인 노트

## 위치

기존 `/heart` 페이지 하단에 "혈압" 섹션 추가.

## 레이아웃

```
/heart 페이지
├── 기존: 안정시 심박 + HRV (유지)
│
├── [혈압] ← 신규 섹션
│   ├── 최근 수치 카드 (수축기/이완기/맥박 + 카테고리 배지)
│   ├── 혈압 추세 차트 (30일/90일 토글)
│   │   - 수축기: 실선 (빨강)
│   │   - 이완기: 실선 (파랑)
│   │   - 카테고리 밴드: 배경색 (정상=초록, 고정상=노랑, 1단계=주황, 2단계=빨강)
│   │   - 정상 범위 120/80 기준선
│   └── 상관관계 차트 (2×2 그리드)
│       ├── 혈압 vs 수면 점수
│       ├── 혈압 vs 평균 스트레스
│       ├── 혈압 vs 바디배터리 (기상)
│       └── 혈압 vs 안정시 심박
│
└── 기존: 심박 추세 (유지)
```

## 상관관계 차트

이중 Y축 라인 차트 (ComposedChart):
- 왼쪽 Y: 수축기 혈압 (mmHg)
- 오른쪽 Y: 비교 지표 (점수/bpm 등)
- X: 날짜

데이터가 부족하면(< 7일) "데이터 수집 중" 메시지 표시.

- 고정상: #f59e0b20
