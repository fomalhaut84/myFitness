# [M2-4] 수면 상세 페이지 신규

## 목적

개별 수면 기록의 상세 분석 페이지. 수면 목록에서 클릭하면 해당 날짜의 상세 데이터 표시.

## 요구사항

- [ ] `/sleep/[date]` 상세 페이지
- [ ] 수면 요약 (총 시간, 점수, 시작/종료, 배터리 변화량)
- [ ] 수면 단계 수평 바 (깊은/얕은/REM/깨어남)
- [ ] 수면 점수 세부 (총시간, 스트레스, 깨어남횟수, REM%, 깊은수면%, 얕은수면%, 뒤척임)
- [ ] 바이탈 지표 (SpO2, 호흡수 평균/최저/최고, 수면 스트레스, 안정시 심박, HRV)
- [ ] AI 평가 버튼
- [ ] 수면 목록에서 상세 페이지로 링크

## 기술 설계

### 수정/생성 파일
- `src/app/sleep/[date]/page.tsx` — 상세 페이지 (신규)
- `src/app/sleep/sleep-client.tsx` — 목록 항목에 링크 추가
