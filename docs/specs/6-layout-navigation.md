# [Phase 2] 레이아웃 + 네비게이션

## 목적

전체 앱의 레이아웃 프레임(사이드바 + 메인 콘텐츠)과 페이지 네비게이션 구현.
다크 테마 기본, 모바일 반응형.

## 요구사항

- [ ] 사이드바 네비게이션 (데스크톱: 고정, 모바일: 햄버거 메뉴)
- [ ] 네비게이션 항목: 대시보드, 러닝/활동, 수면, 심박, 체성분
- [ ] 현재 페이지 활성 표시
- [ ] 다크 테마 기본 적용
- [ ] 모바일 반응형 (md 브레이크포인트)
- [ ] 앱 타이틀 "myFitness" + 피트니스 아이콘

## 기술 설계

### 컴포넌트 구조

```
src/components/
├── layout/
│   ├── Sidebar.tsx      # 사이드바 네비게이션
│   └── MainLayout.tsx   # 사이드바 + 메인 콘텐츠 래퍼
```

### 네비게이션 항목

| 경로 | 라벨 | 아이콘 |
|---|---|---|
| / | 대시보드 | LayoutDashboard |
| /activities | 러닝/활동 | Activity |
| /sleep | 수면 | Moon |
| /heart | 심박 | Heart |
| /body | 체성분 | Scale |

### 반응형 전략

- `md` 이상: 사이드바 고정 (w-60), 메인 콘텐츠 ml-60
- `md` 미만: 사이드바 숨김, 상단에 햄버거 버튼 → 오버레이 사이드바

### 아이콘

SVG 직접 사용 (외부 아이콘 라이브러리 미설치, 필요한 것만 인라인)
