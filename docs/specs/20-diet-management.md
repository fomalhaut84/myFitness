# [Phase 5] 다이어트 관리

## 목적

체중 목표 설정, 식단 텍스트 입력 → AI 칼로리 추정, 칼로리 밸런스 시각화.

## 요구사항

- [ ] 체중 목표 설정 (UserProfile.targetWeight)
- [ ] 식단 기록 입력 (자유 텍스트, 끼니 선택)
- [ ] AI 칼로리 추정 (Claude로 텍스트 → kcal 변환)
- [ ] 일일 칼로리 밸런스 (소모 vs 섭취)
- [ ] 체중 목표 대비 진행 현황

## 기술 설계

`/body` 페이지에 다이어트 섹션 추가.
FoodLog 모델 활용 (이미 DB 스키마에 정의됨).
