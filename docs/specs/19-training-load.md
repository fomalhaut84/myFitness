# [Phase 5] 트레이닝 로드 분석

## 목적

주간 운동 볼륨 추세와 오버트레이닝 위험 지표를 시각화.

## 요구사항

- [ ] 주간 운동 볼륨 바 차트 (최근 8주, 거리 or 시간)
- [ ] 주간 볼륨 증가율 표시 (10% 이하 권장)
- [ ] 오버트레이닝 위험 지표 (안정시HR 상승 + HRV 하락 + 수면 질 저하 조합)
- [ ] 회복 상태 요약 카드

## 기술 설계

`/activities` 페이지 하단 또는 `/lifestyle` 페이지에 통합.

### 오버트레이닝 위험 판단
- 안정시HR이 7일 평균 대비 5bpm+ 상승
- HRV가 7일 평균 대비 20%+ 하락
- 수면 점수가 7일 평균 대비 15+ 하락
- 3개 중 2개 이상 해당 → 위험
