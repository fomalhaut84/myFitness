# [Phase 2] 심박/HRV + 체성분 페이지

## 목적

심박수/HRV 추세와 체중/체지방 추세를 시각화하는 페이지 2개 구현.

## 요구사항

### 심박 페이지 (`/heart`)
- [ ] 오늘 안정시 심박 카드
- [ ] 안정시 심박 30일 추세 라인 차트
- [ ] HRV 30일 추세 라인 차트
- [ ] 최근 14일 심박 기록 목록

### 체성분 페이지 (`/body`)
- [ ] 최근 체중 카드
- [ ] 체중 30일 추세 라인 차트
- [ ] 체지방률 추세 (데이터 있을 때만)
- [ ] 최근 14일 체중 기록 목록

## 기술 설계

### 컬러
- 안정시 심박: #ef4444 (레드)
- HRV: #f59e0b (앰버)
- 체중: #60a5fa (블루)
- 체지방: #a78bfa (퍼플)

### 컴포넌트 구조

```
src/components/heart/
└── HRTrendChart.tsx     # 안정시HR/HRV 라인 차트 (재사용)
src/components/body/
└── WeightChart.tsx      # 체중/체지방 라인 차트
```
