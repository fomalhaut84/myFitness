# [Phase 3] 생활 패턴 분석 페이지

## 목적

주간/월간 단위로 운동 꾸준함, 수면 규칙성, 휴식일 패턴을 분석하는 페이지.

## 요구사항

- [ ] 주간 활동 요약 (이번 주 운동 횟수, 총 거리, 휴식일)
- [ ] 월간 활동 캘린더 히트맵 (운동한 날 표시)
- [ ] 운동 꾸준함 점수 (최근 4주 중 운동한 주 비율)
- [ ] 수면 규칙성 (최근 14일 취침/기상 시간 분포)
- [ ] 주간 비교 (이번 주 vs 지난 주)

## 기술 설계

### 페이지

`/lifestyle` (사이드바에 이미 있음 — 현재 빈 페이지)

### 컴포넌트

```
src/components/lifestyle/
├── WeeklyActivitySummary.tsx   # 이번 주 운동 요약
├── MonthlyHeatmap.tsx          # 월간 활동 캘린더
├── ConsistencyScore.tsx        # 꾸준함 점수
└── SleepRegularity.tsx         # 수면 규칙성
```

### 계산 로직

**꾸준함 점수**: 최근 28일 중 활동이 있는 날 / 28 × 100
**휴식일**: 활동이 없는 날 (의도적 회복)
**수면 규칙성**: 취침 시간의 표준편차 (낮을수록 규칙적)
