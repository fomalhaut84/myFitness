# [Phase 2] 수면 페이지

## 목적

수면 데이터를 시각화하는 페이지. 수면 단계별 비율, 수면 점수 추세 표시.

## 요구사항

- [ ] 어젯밤 수면 요약 카드 (총 수면, 점수, 단계별 시간)
- [ ] 수면 단계 바 차트 (깊은/얕은/REM/깨어남 비율)
- [ ] 수면 점수 30일 추세 라인 차트
- [ ] 수면 기록 목록 (최근 14일)

## 기술 설계

### 컴포넌트 구조

```
src/components/sleep/
├── SleepSummary.tsx      # 어젯밤 요약
├── SleepStagesBar.tsx    # 단계별 수평 바
└── SleepScoreChart.tsx   # 30일 점수 추세
```

### 수면 단계 컬러

- 깊은 수면: #3b82f6 (블루)
- 얕은 수면: #93c5fd (라이트 블루)
- REM: #a78bfa (퍼플)
- 깨어남: #525252 (그레이)

### 포맷 규칙

- 수면 시간: Xh Xm (분 → 시간:분 변환)
- 수면 점수: 0~100 정수
- 시작/종료: HH:MM
