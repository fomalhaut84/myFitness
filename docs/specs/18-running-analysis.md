# [Phase 5] 러닝 실력 분석

## 목적

러닝 페이스, HR존 분포, VO2max 추세를 분석하고 훈련 존 추천을 제공하는 페이지.

## 요구사항

- [ ] 러닝 페이스 추세 (30일 라인 차트)
- [ ] HR존 분포 (최근 러닝의 HR존별 시간 비율)
- [ ] VO2max 추세 (라인 차트)
- [ ] 최근 러닝 성과 테이블 (날짜, 거리, 페이스, 평균HR, TE)
- [ ] 훈련 존 가이드 (이지/템포/인터벌 기준)

## 기술 설계

기존 `/activities` 페이지에 "러닝 분석" 탭 추가 또는 별도 섹션.

### HR존 계산 (최대 심박 기준)
- Zone 1 (50-60%): 회복
- Zone 2 (60-70%): 이지런
- Zone 3 (70-80%): 에어로빅
- Zone 4 (80-90%): 템포/역치
- Zone 5 (90-100%): 인터벌/VO2max

최대 심박 추정: 220 - 나이 (또는 실측 maxHR)
